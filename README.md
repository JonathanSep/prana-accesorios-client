## üì¶ Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/JonathanSep/prana-accesorios-client.git](https://github.com/JonathanSep/prana-accesorios-client.git)
    cd prana-accesorios-client
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Environment Variables**
    Create a `.env` file in the root directory to connect to your backend:
    ```env
    VITE_API_URL=[https://prana-accesorios-api.onrender.com/api](https://prana-accesorios-api.onrender.com/api)
    ```

4.  **Run the application**
    ```bash
    npm run dev
    ```
